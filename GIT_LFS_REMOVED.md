## Почему это важно

Git LFS (Large File Storage) может вызывать проблемы с загрузкой файлов на GitHub Pages, особенно для больших JSON-файлов (50MB в нашем случае).

## Как теперь работает загрузка данных

1. **Основной вариант**: Загрузка из split-файлов (`data/YYYY-MM-DD/matches_part_*.json`)
2. **Резервный вариант**: Загрузка из combined-файла (`all_top3_data.json`)

## Файлы для использования

### Основной файл:
- `index.html` - Полная версия с умной системой загрузки

### Альтернативный файл:
- `index_simple.html` - Упрощенная версия для быстрого тестирования

### Инструменты для отладки:
- `debug_loading.html` - Полная диагностика проблем с загрузкой
- `test_split_loading.html` - Тестирование split-файлов
- `test_loading.html` - Тестирование системы загрузки

## Как проверить, что всё работает

1. Запустите локальный сервер: `python -m http.server 8000`
2. Откройте в браузере: `http://localhost:8000/index.html`
3. Если есть проблемы, используйте `debug_loading.html` для диагностики
